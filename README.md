## how to run

> download Android Studio and an emulator of your choice

> run any emulator on vscode
> go to root dir, run 'flutter run'
> ez

The questions are based on a combination of GAD-7 (Generalized Anxiety Disorder) and PHQ-9 (Patient Health Questionnaire) scales, which are commonly used for mental health screening. Each question asks about symptoms over the past two weeks.
The options for each question (1-5) will represent:

- Not at all
- Several days
- More than half the days
- Nearly every day
- Every day
